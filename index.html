Jogo do Círculo

Pontuação: 0

Instruções

O objetivo do jogo é clicar no círculo vermelho que aparece em posições aleatórias na tela. Cada clique no círculo aumenta sua pontuação em 1 ponto. Tente obter a maior pontuação possível!

Sobre

Este jogo foi criado como um exemplo de projeto para aprender HTML, CSS e JavaScript. Divirta-se jogando!
